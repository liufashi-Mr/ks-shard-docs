## KeyValuesModal

KeyValuesModal 是一个键值对编辑模态框组件，用于编辑标签和注解等键值对数据。

### API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| title | 标题 | `string` | - |
| fieldName | 字段名 | `FormItemProps['name']` | - |
| initialValues | 初始值 | `OriginData` | - |
| visible | 是否可见 | `boolean` | - |
| onOk | 确认回调 | `(value: OriginData) => void` | - |
| onCancel | 取消回调 | `() => void` | - |
| confirmLoading | 确认按钮加载状态 | `boolean` | - |
| hideFooter | 隐藏底部 | `boolean` | - |
| cancelText | 取消按钮文本 | `string` | - |
| hiddenKeys | 隐藏的键 | `string[]` | - |
| readOnlyKeys | 只读的键 | `string[]` | - |
| okText | 确认按钮文本 | `string` | - |
