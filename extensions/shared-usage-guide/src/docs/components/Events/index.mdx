### API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| detail | 资源详细信息 | `T` | - |
| module | 模块名称 | `string` | - |
| showTitle | 是否显示标题 | `boolean` | - |
