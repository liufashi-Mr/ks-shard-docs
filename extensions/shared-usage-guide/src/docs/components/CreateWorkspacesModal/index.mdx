### API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| visible | 是否可见 | `boolean` | - |
| width | 模态框宽度 | `number` | - |
| confirmLoading | 确认按钮加载状态 | `boolean` | - |
| onOk | 确认回调 | `(data: WorkspaceFormValues) => void` | - |
| onCancel | 取消回调 | `() => void` | - |
