NavTitle 是一个导航标题组件，用于显示页面或模块的标题信息。

### API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| icon | 图标 | `ReactNode` | - |
| title | 标题 | `string` | - |
| subtitle | 副标题 | `string` | - |
| onClick | 点击事件 | `() => void` | - |
