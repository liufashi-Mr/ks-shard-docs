### API

| 参数        | 说明           | 类型         | 默认值 |
| ----------- | -------------- | ------------ | ------ |
| icon        | 图标           | `any`        | -      |
| title       | 标题           | `any`        | -      |
| description | 描述           | `any`        | -      |
| className   | 自定义类名     | `string`     | -      |
| ellipsis    | 是否显示省略号 | `boolean`    | -      |
| extra       | 额外内容       | `any`        | -      |
| onClick     | 点击事件       | `() => void` | -      |
| tooltipIcon | 提示图标       | `any`        | -      |
