## Bar

用于展示进度或占比的条形组件。

### Usage

<DemoBlock title="Bar 基本用法" description="传入数值与文本展示占比">
  <Basic />
</DemoBlock>

### API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| value | 百分比数值 0-100 | `number` | - |
| text | 左侧文本 | `string` | - |
| rightText | 右侧文本 | `string` | - |
| className | 自定义类名 | `string` | - |
