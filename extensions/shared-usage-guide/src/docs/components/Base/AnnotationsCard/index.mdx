## AnnotationsCard

### Usage

AnnotationsCard 组件用于展示注解对象，并支持隐藏某些键，一般用在 annotations 字段上。

### API

| 参数        | 说明            | 类型                        | 默认值 |
| ----------- | --------------- | --------------------------- | ------ |
| className   | 自定义类名      | string                      | -      |
| annotations | 注解对象        | `{ [key: string]: string }` | -      |
| hiddenKeys  | 隐藏的 key 列表 | string[]                    | -      |
