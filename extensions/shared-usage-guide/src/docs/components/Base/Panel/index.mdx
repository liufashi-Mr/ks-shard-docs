## Panel

### Usage

### API

| 参数 | 说明 | 类型 |
| --- | --- | --- |
| title | 面板标题 | `string` |
| loading | 加载中 | `boolean` |
| extras | 右上角区域 | `any` |
| children | 面板内容 | `ReactNode` |
| className | 自定义类名 | `string` |
