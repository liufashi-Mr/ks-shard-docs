import Basic from './Basic';

## CopyButton

用于一键复制文本的按钮。

### Usage

<DemoBlock title="CopyButton 基本用法" description="点击按钮复制指定文本">
  <Basic />
</DemoBlock>

### API

| 参数 | 说明 | 类型 |
| --- | --- | --- |
| value | 需要复制的文本 | `string` |
| label | 按钮文本 | `ReactNode` |
| timeout | 成功提示停留时间(ms) | `number` |
| iconProps | 图标属性（含 hoverVariant） | `IconProps` |
| styles | 自定义样式 | `{ icon?: CSSProperties }` |
| hoverStyles | 悬停样式 | `{ icon?: CSSProperties }` |
