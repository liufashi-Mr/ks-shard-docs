## Icon

Icon 是一个图标组件，支持多种图标类型、颜色变体和尺寸设置。基于[Kube Design Icons](https://kube.design/zh/Icons/Icons), 将文档中的 PascalCase转为kebab-case 给 Icon 组件的 name 属性

### API

| 参数    | 说明     | 类型                                        | 默认值 |
| ------- | -------- | ------------------------------------------- | ------ |
| name    | 图标名称 | `string`                                    | -      |
| color   | 图标颜色 | `string`                                    | -      |
| variant | 颜色变体 | `'dark' \| 'light' \| 'coloured' \| string` | -      |
| size    | 图标尺寸 | `number`                                    | -      |
