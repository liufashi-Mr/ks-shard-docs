PasswordTip 是一个密码强度提示组件，用于显示密码强度要求和验证结果。

### API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| password | 密码字符串 | `string` | - |
| className | 自定义类名 | `string` | - |
| hasProgress | 是否显示进度条 | `boolean` | - |
