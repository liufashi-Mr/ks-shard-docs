### API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| currentTab | 当前标签页 | `string` | - |
| banner | 横幅配置 | `BannerType` | - |
| table | 表格配置 | `Table` | - |
| tabs | 标签页配置 | `ListTabProps[]` | - |
| store | 数据存储 | `any` | - |
| hasNamespaceSelector | 是否有命名空间选择器 | `boolean` | - |
| onNamespaceChange | 命名空间变化回调 | `(name: string) => void` | - |
| getTableUrl | 获取表格URL | `(namespace: string) => string` | - |
