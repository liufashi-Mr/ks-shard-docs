### API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| headerTitle | 头部标题 | `ReactNode` | - |
| headerDescription | 头部描述 | `ReactNode` | - |
| headerIcon | 头部图标 | `ReactNode` | - |
| sheetContentProps | 侧边栏内容属性 | `SheetContentProps` | - |
