### API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| visible | 是否可见 | `boolean` | - |
| name | 资源名称 | `string` | - |
| type | 资源类型 | `string` | - |
| confirmLoading | 确认按钮加载状态 | `boolean` | - |
| onOk | 确认回调 | `() => void` | - |
| onCancel | 取消回调 | `() => void` | - |
| modalProps | 模态框属性 | `ModalProps` | - |
