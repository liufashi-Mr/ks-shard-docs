### API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| websocketUrl | WebSocket URL | `string` | - |
| terminalOptions | 终端选项 | `TerminalOptions` | - |
| initText | 初始文本 | `string` | - |
| isEdgeNode | 是否为边缘节点 | `boolean` | - |
| isLoading | 是否加载中 | `boolean` | - |
| uploadUrl | 上传 URL | `string` | - |
| downloadUrl | 下载 URL | `string` | - |
