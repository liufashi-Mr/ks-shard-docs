### API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| collapsed | 是否折叠 | `boolean` | - |
| title | 标题 | `string` | - |
| subtitle | 副标题 | `string` | - |
| content | 内容 | `ReactNode` | - |
