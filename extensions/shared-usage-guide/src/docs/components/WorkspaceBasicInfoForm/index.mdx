### API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| initialValues | 初始值 | `OriginalWorkspace` | - |
| isHideManagerFiled | 是否隐藏管理员字段 | `boolean` | - |
| manager | 管理员 | `string` | - |
| onOk | 确认回调 | `(value: WorkspaceFormValues) => void` | - |
