### API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| hideFilters | 隐藏筛选器 | `boolean` | - |
| hideSettingMenu | 隐藏设置菜单 | `boolean` | - |
| toolbarLeft | 左侧工具栏 | `ReactNode` | - |
| toolbarRight | 右侧工具栏 | `ReactNode` | - |
| batchActions | 批量操作 | `ReactNode` | - |
| simpleSearch | 简单搜索 | `boolean` | - |
| instance | 表格实例 | `TableInstance<T>` | - |
| placeholder | 占位符 | `string` | - |
| suggestions | 建议 | `Suggestions` | - |
| onFilterInputChange | 筛选输入变化 | `(value: any) => void` | - |
| refetch | 重新获取 | `any` | - |
