## Skeletons

Skeletons 是一组骨架屏组件，用于在数据加载时显示占位内容。

### Usage

#### TableSkeleton

`TableSkeleton` 是一个表格骨架屏组件，用于在表格数据加载时显示占位内容。

### API

#### TableSkeleton

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| row | 行数 | `number` | - |
| hasCreateButton | 是否有创建按钮 | `boolean` | - |
| hasToolbar | 是否有工具栏 | `boolean` | - |
