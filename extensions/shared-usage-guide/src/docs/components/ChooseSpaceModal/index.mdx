### API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| visible | 是否可见 | `boolean` | - |
| onOk | 确认回调 | `(data: RejectFormData) => void` | - |
| onCancel | 取消回调 | `() => void` | - |
| defaultVal | 默认值 | `{ namespace?: string; workspace?: string; cluster?: string; }` | - |
