## DetailPage

DetailPage 是一个详情页面布局组件，用于展示资源的详细信息，支持标签页、侧边栏等功能。

### API

| 参数      | 说明             | 类型                                                                          | 默认值 |
| --------- | ---------------- | ----------------------------------------------------------------------------- | ------ |
| tabs      | 标签页配置       | `Tab[]`                                                                       | -      |
| nav       | 导航内容         | `ReactNode`                                                                   | -      |
| activeTab | 当前激活的标签页  | `string`                                                                      | -      |
| sideProps | 侧边栏属性       | `SideBarProps<T>`                                                             | -      |
| store     | 数据存储         | `any`                                                                         | -      |
| data      | 详情数据         | `T`                                                                           | -      |
| params    | 路径参数         | `PathParams`                                                                  | -      |
| isLoading | 加载状态         | `boolean`                                                                     | -      |
| authKey   | 权限键           | `string`                                                                      | -      |
| className | 自定义类名       | `string`                                                                      | -      |
| padding   | 内边距           | `number`                                                                      | -      |
| replica   | 副本信息         | `ReactNode \| ((detail: T) => ReactNode)`                                     | -      |
| customNav | 自定义导航       | `(props: { tabs?: Tab[], nav?: ReactNode, activeTab?: string }) => ReactNode` | -      |

### Tab 配置

| 参数      | 说明       | 类型        | 默认值 |
| --------- | ---------- | ----------- | ------ |
| name      | 标签页名称 | `string`    | -      |
| title     | 标签页标题 | `string`    | -      |
| component | 标签页组件 | `ReactNode` | -      |
| authKey   | 权限键     | `string`    | -      |
