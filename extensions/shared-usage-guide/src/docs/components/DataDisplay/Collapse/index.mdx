Collapse 是一个折叠面板组件，支持展开/收起功能和自定义图标。

### API

| 参数             | 说明             | 类型                      | 默认值  |
| ---------------- | ---------------- | ------------------------- | ------- |
| title            | 标题内容         | `ReactNode`               | -       |
| icon             | 自定义图标       | `ReactNode`               | -       |
| isOpen           | 是否展开         | `boolean`                 | -       |
| defaultOpen      | 默认是否展开     | `boolean`                 | `false` |
| onChange         | 展开状态变化回调 | `(open: boolean) => void` | -       |
| children         | 子内容           | `ReactNode`               | -       |
| className        | 自定义类名       | `string`                  | -       |
| headerClassName  | 头部类名         | `string`                  | -       |
| contentClassName | 内容类名         | `string`                  | -       |
