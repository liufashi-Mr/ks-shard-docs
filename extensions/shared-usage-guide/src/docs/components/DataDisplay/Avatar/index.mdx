## Avatar

Avatar 是一个头像组件，用于显示用户、项目或资源的头像信息，支持图标、标题和描述。

### Usage

`Avatar`

### API

| 参数            | 说明         | 类型                  | 默认值 |
| --------------- | ------------ | --------------------- | ------ |
| icon            | 图标         | `string \| ReactNode` | -      |
| iconSize        | 图标大小     | `number`              | -      |
| to              | 链接地址     | `string`              | -      |
| title           | 标题         | `string \| ReactNode` | -      |
| description     | 描述         | `string \| ReactNode` | -      |
| className       | 自定义类名   | `string`              | -      |
| isMultiCluster  | 是否为多集群 | `boolean`             | -      |
| record          | 记录数据     | `any`                 | -      |
| module          | 模块名称     | `string`              | -      |
| showDescription | 是否显示描述 | `boolean`             | -      |
