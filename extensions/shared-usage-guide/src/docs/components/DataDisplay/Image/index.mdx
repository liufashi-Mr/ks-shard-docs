## Image

Image 是一个图片组件，支持图片显示、加载状态和降级处理，常用于显示容器镜像、头像等。

### API

| 参数        | 说明                 | 类型      | 默认值  |
| ----------- | -------------------- | --------- | ------- |
| src         | 图片地址             | `string`  | -       |
| iconSize    | 图标尺寸             | `number`  | -       |
| iconLetter  | 降级显示的文字       | `string`  | -       |
| className   | 自定义类名           | `string`  | -       |
| isBase64Str | 是否为 Base64 字符串 | `boolean` | `false` |
| canLoading  | 是否显示加载状态     | `boolean` | -       |
