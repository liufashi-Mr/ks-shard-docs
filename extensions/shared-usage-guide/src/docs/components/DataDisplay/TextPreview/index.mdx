TextPreview 是一个文本预览组件，用于展示文本文件，支持多种文件格式和代码高亮。

### API

| 参数          | 说明           | 类型                                                       | 默认值  |
| ------------- | -------------- | ---------------------------------------------------------- | ------- |
| files         | 文件内容对象   | `Record<string, string>`                                   | -       |
| type          | 文件类型       | `string`                                                   | -       |
| hideToolbar   | 是否隐藏工具栏 | `boolean`                                                  | `false` |
| editorOptions | 编辑器配置选项 | `Partial<CodeEditorProps>` mode 目前仅支持`yaml`和`groovy` | -       |
