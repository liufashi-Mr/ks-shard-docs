Markdown 是一个 Markdown 渲染组件，支持 GitHub Flavored Markdown (GFM) 和主题定制。

### API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| children | Markdown 文本内容 | `string` | - |
| isSupportGFM | 是否支持 GFM | `boolean` | `false` |
| themeName | 主题名称 | `'global-styles' \| 'github-light'` | `'global-styles'` |
| classNames | 自定义类名 | `{ root?: string, markdown?: string }` | - |
| styles | 自定义样式 | `{ root?: CSSProperties }` | - |
