### API

#### TinyArea

| 参数       | 说明       | 类型                                         | 默认值 |
| ---------- | ---------- | -------------------------------------------- | ------ |
| width      | 宽度       | `number \| string`                           | -      |
| height     | 高度       | `number \| string`                           | -      |
| title      | 标题       | `string`                                     | -      |
| unit       | 单位       | `string`                                     | -      |
| data       | 数据       | `Record<string, string \| number \| null>[]` | -      |
| bgColor    | 背景色     | `string`                                     | -      |
| areaColors | 面积图颜色 | `any[]`                                      | -      |
| titleNode  | 标题节点   | `ReactNode`                                  | -      |
| areaNode   | 面积图节点 | `ReactNode`                                  | -      |
| darkMode   | 暗色模式   | `boolean`                                    | -      |
