### API

| 参数         | 说明       | 类型                                                                                                                         | 默认值 |
| ------------ | ---------- | ---------------------------------------------------------------------------------------------------------------------------- | ------ |
| children     | 子元素     | `ReactNode`                                                                                                                  | -      |
| isLoading    | 是否加载中 | `boolean`                                                                                                                    | -      |
| loading      | 加载组件   | `ReactNode`                                                                                                                  | -      |
| isRefreshing | 是否刷新中 | `boolean`                                                                                                                    | -      |
| isEmpty      | 是否为空   | `boolean`                                                                                                                    | -      |
| empty        | 空状态组件 | `ReactNode`                                                                                                                  | -      |
| classNames   | 类名配置   | `{ container?: string; content?: string; loadingWrapper?: string; refreshingWrapper?: string; }`                             | -      |
| styles       | 样式配置   | `{ container?: CSSProperties; content?: CSSProperties; loadingWrapper?: CSSProperties; refreshingWrapper?: CSSProperties; }` | -      |
