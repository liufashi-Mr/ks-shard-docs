### API

#### List 组件

| 参数      | 说明       | 类型        | 默认值 |
| --------- | ---------- | ----------- | ------ |
| className | 自定义类名 | `string`    | -      |
| children  | 子元素     | `ReactNode` | -      |

#### ListItem 组件

| 参数        | 说明       | 类型                  | 默认值 |
| ----------- | ---------- | --------------------- | ------ |
| icon        | 图标       | `string`              | -      |
| image       | 图片       | `string`              | -      |
| title       | 标题       | `string \| ReactNode` | -      |
| status      | 状态       | `string \| ReactNode` | -      |
| description | 描述       | `string \| ReactNode` | -      |
| extras      | 额外信息   | `ReactNode`           | -      |
| details     | 详细信息   | `ReactNode`           | -      |
| operations  | 操作按钮   | `ReactNode`           | -      |
| onDelete    | 删除回调   | `() => void`          | -      |
| onEdit      | 编辑回调   | `() => void`          | -      |
| onClick     | 点击回调   | `() => void`          | -      |
| className   | 自定义类名 | `string`              | -      |
| titleClass  | 标题类名   | `string`              | -      |
