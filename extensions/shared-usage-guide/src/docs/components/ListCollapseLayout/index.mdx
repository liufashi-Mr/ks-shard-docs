## ListCollapseLayout

ListCollapseLayout 是一个可折叠的列表布局组件，用于展示可折叠的列表页面。

### API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| children | 子元素 | `ReactNode` | - |
| navMenu | 导航菜单 | `ReactNode` | - |
| showCollapsedBtn | 是否显示折叠按钮 | `boolean` | - |
| switcher | 切换器配置 | `Switcher` | - |
| title | 标题 | `ReactNode` | - |
