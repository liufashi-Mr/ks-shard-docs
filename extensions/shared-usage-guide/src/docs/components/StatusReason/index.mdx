## StatusReason

StatusReason 是一个状态原因组件，用于显示资源状态和失败原因，常用于 Pod 和容器状态展示。

### API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| data | 状态数据 | `any` | - |
| reason | 状态原因 | `string` | - |
| status | 状态 | `string` | - |
| type | 类型 | `string` | - |
| hasTip | 是否显示提示 | `boolean` | - |
| onShowEvents | 显示事件回调 | `() => void` | - |
