## TimeSelector

TimeSelector 是一个时间选择器组件，用于选择时间范围和刷新间隔，常用于监控和日志查看场景。

### API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| arrowIcon | 箭头图标 | `ReactNode` | - |
| dark | 深色主题 | `boolean` | `false` |
| step | 刷新间隔 | `string` | - |
| times | 时间倍数 | `number` | - |
| renderCustom | 是否渲染自定义选项 | `boolean` | - |
| showStep | 是否显示刷新间隔 | `boolean` | - |
| onChange | 时间变化回调 | `(val: Omit<ValueType, 'lastTime'>) => void` | - |
| onToggle | 切换回调 | `(toggled: boolean) => void` | - |
| timeOps | 时间选项 | `string[]` | - |
| zIndex | 层级 | `number` | - |
