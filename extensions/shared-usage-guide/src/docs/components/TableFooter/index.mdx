### API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| instance | 表格实例 | `TableInstance` | - |
| totalCount | 总数量 | `number` | - |
| isShowSetPageSize | 是否显示页面大小设置 | `boolean` | - |
| isShowDivider | 是否显示分割线 | `boolean` | - |
| isShowTotalCount | 是否显示总数量 | `boolean` | - |
| isShowPagination | 是否显示分页 | `boolean` | - |
| className | 类名 | `string` | - |
| style | 样式 | `CSSProperties` | - |
| placement | 位置 | `any` | - |
