### API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| cluster | 集群名称 | `string` | - |
| workspace | 工作空间 | `string` | - |
| enableActions | 启用操作 | `Record<string, string[]>` | - |
| onSelect | 选择回调 | `(url?: string) => void` | - |
