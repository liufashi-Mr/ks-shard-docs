## CollapsedBtn

CollapsedBtn 是一个折叠按钮组件，用于控制侧边栏的展开和收起状态。

### Usage

### API

| 参数             | 说明         | 类型                           | 默认值 |
| ---------------- | ------------ | ------------------------------ | ------ |
| collapsed        | 是否折叠     | `boolean`                      | -      |
| onClickCollapsed | 点击折叠回调 | `(collapsed: boolean) => void` | -      |
