### API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| title | 页面标题 | `ReactNode` | - |
| tipsIconProps | 提示图标属性 | `{ tipsText?: string; imgBase64?: string; iconName?: string; maxWidth?: number; }` | - |
| tabs | 标签页配置 | `PageHeaderTab[]` | - |
| headerRight | 右侧内容 | `ReactNode` | - |
| activeTab | 当前激活标签 | `string` | - |
| type | 标签类型 | `'link' \| 'button'` | - |
| onTabChange | 标签切换回调 | `(tab: string) => void` | - |
| wrapperT | 是否包装 T | `boolean` | - |
| toParent | 是否跳转到父级 | `boolean` | - |
| padding | 内边距 | `number` | - |
