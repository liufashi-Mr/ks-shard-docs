## PodIndicator

PodIndicator 是一个 Pod 状态指示器组件，用于显示 Pod 的详细状态信息，包括容器状态、重启次数等。

### Usage

`PodIndicator`

### API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| module | 模块名称 | `string` | - |
| cluster | 集群名称 | `string` | - |
| record | Pod 记录数据 | `FormattedPod` | - |
