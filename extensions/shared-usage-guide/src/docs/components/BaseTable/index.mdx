### API

| 参数       | 说明     | 类型          | 默认值 |
| ---------- | -------- | ------------- | ------ |
| dataSource | 数据源   | `T[]`         | -      |
| loading    | 加载状态 | `boolean`     | -      |
| columns    | 列配置   | `Column<T>[]` | -      |
