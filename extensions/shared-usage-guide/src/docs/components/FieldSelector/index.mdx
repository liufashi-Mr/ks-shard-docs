### API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| value | 当前值 | `string` | - |
| onChange | 变化回调 | `(value: string) => void` | - |
| defaultValue | 默认值 | `string` | - |
| Item | 项组件 | `ComponentType` | - |
| list | 数据列表 | `T[]` | - |
| renderPlaceholder | 渲染占位符 | `() => ReactNode` | - |
| rowKey | 行键 | `string` | - |
