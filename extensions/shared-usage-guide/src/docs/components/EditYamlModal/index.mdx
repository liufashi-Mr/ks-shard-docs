### API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| yaml | YAML 内容 | `string` | - |
| detail | 详细信息 | `Record<string, any>` | - |
| visible | 是否可见 | `boolean` | - |
| onOk | 确认回调 | `(yaml: string) => void` | - |
| onCancel | 取消回调 | `() => void` | - |
| onChange | 变化回调 | `(yaml: string) => void` | - |
| readOnly | 只读模式 | `boolean` | - |
| confirmLoading | 确认按钮加载状态 | `boolean` | - |
