## InfoConfirm

InfoConfirm 是一个信息确认模态框组件，用于显示信息并获取用户确认。

### API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| visible | 是否可见 | `boolean` | - |
| title | 标题 | `ReactNode` | - |
| content | 内容 | `ReactNode` | - |
| onOk | 确认回调 | `() => void` | - |
| onCancel | 取消回调 | `() => void` | - |
