### API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| visible | 是否可见 | `boolean` | - |
| detail | 应用详情 | `any` | - |
| onCancel | 取消回调 | `() => void` | - |
| workspace | 工作空间 | `string` | - |
| cluster | 集群名称 | `string` | - |
