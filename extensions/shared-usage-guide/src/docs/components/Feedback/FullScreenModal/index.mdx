### API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| visible | 是否可见 | `boolean` | - |
| onCancel | 取消回调 | `() => void` | - |
| title | 标题 | `ReactNode` | - |
| children | 子元素 | `ReactNode` | - |
