### API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| visible | 是否可见 | `boolean` | - |
| type | 资源类型 | `string` | - |
| resource | 资源名称 | `string` | - |
| title | 标题 | `string` | - |
| desc | 描述 | `string` | - |
| app | 应用名称 | `string` | - |
| isSubmitting | 是否提交中 | `boolean` | - |
| onCancel | 取消回调 | `() => void` | - |
| onOk | 确认回调 | `() => void` | - |
| confirmLoading | 确认按钮加载状态 | `boolean` | - |
