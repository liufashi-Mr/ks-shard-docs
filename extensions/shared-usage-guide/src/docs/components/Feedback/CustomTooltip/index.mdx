### API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| renderLabel | 自定义标签渲染函数 | `(res: unknown) => {}` | - |
| active | 是否激活 | `boolean` | - |
| payload | 数据载荷 | `Record<string, any>[]` | - |
| usageData | 使用数据 | `{ [x: string]: string \| number \| null }[]` | - |
| totalData | 总数据 | `{ [x: string]: string \| number \| null }[]` | - |
| tailText | 尾部文本 | `string` | - |
| alert | 告警信息 | `Record<string, any>` | - |
| isTranslate | 是否翻译 | `boolean` | - |
| label | 标签 | `string` | - |
| unit | 单位 | `string` | - |
