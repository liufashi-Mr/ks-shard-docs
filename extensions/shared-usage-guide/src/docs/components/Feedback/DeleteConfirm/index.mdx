### API

| 参数           | 说明         | 类型                                 | 默认值 |
| -------------- | ------------ | ------------------------------------ | ------ |
| type           | 资源类型     | `string`                             | -      |
| resource       | 要删除的资源 | `Resource`                           | -      |
| deleteCluster  | 是否删除集群 | `boolean`                            | -      |
| app            | 应用信息     | `any`                                | -      |
| title          | 标题         | `ReactNode`                          | -      |
| tip            | 提示信息     | `string`                             | -      |
| desc           | 描述         | `ReactNode`                          | -      |
| confirmLoading  | loading      | `boolean`                            | -      |
| onOk           | 确认回调     | `(data: any[] \| undefined) => void` | -      |
| onSuccess      | 成功回调     | `() => void`                         | -      |
| onCancel       | 取消回调     | `() => void`                         | -      |
