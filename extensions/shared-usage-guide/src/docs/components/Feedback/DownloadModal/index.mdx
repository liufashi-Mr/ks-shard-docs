### API

| 参数        | 说明     | 类型         | 默认值 |
| ----------- | -------- | ------------ | ------ |
| visible     | 是否可见 | `boolean`    | -      |
| title       | 标题     | `ReactNode`  | -      |
| downloadUrl | 下载 URL | `string`     | -      |
| onCancel    | 取消回调 | `() => void` | -      |
