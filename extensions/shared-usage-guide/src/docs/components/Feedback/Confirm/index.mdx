## Confirm

Confirm 是一个确认对话框组件，用于显示确认信息并获取用户确认。

### API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| titleIconProps | 标题图标属性 | `TitleIconProps` | - |
| titleIcon | 标题图标 | `ReactNode` | - |
| title | 标题 | `ReactNode` | - |
| description | 描述 | `ReactNode` | - |
| visible | 是否可见 | `boolean` | - |
| onOk | 确认回调 | `() => void` | - |
| onCancel | 取消回调 | `() => void` | - |
