### API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| visible | 是否可见 | `boolean` | - |
| detail | 仓库详情 | `RepoData` | - |
| onOk | 确认回调 | `() => void` | - |
| onCancel | 取消回调 | `() => void` | - |
