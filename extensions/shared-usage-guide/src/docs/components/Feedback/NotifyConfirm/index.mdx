### API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| visible | 是否可见 | `boolean` | - |
| width | 宽度 | `number \| string` | - |
| type | 类型 | `string` | - |
| title | 标题 | `string` | - |
| content | 内容 | `ReactNode` | - |
| btns | 自定义按钮 | `ReactNode` | - |
| confirmText | 确认按钮文本 | `string` | - |
| cancelText | 取消按钮文本 | `string` | - |
| isSubmitting | 是否提交中 | `boolean` | - |
| onConfirm | 确认回调 | `() => void` | - |
| onCancel | 取消回调 | `() => void` | - |
