## KubeCtl

KubeCtl 是一个 kubectl 命令执行模态框组件，用于在浏览器中执行 kubectl 命令。

### API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| visible | 是否可见 | `boolean` | - |
| title | 标题 | `string` | - |
| onCancel | 取消回调 | `() => void` | - |
| params | 参数配置 | `{ nodename?: string; cluster?: string; isEdgeNode?: boolean; isClusterQuery?: boolean; }` | - |
| zIndex | 层级 | `number` | - |
