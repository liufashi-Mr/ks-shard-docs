### API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| visible | 是否可见 | `boolean` | - |
| onCancel | 取消回调 | `() => void` | - |
| params | 参数配置 | `{ cluster?: string; }` | - |
| zIndex | 层级 | `number` | - |
