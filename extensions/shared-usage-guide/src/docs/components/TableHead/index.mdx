TableHead 是一个表格头部组件，用于显示表格的列标题和选择控件。

### API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| column | 列配置 | `HeaderGroup<T>` | - |
| selectType | 选择类型 | `'checkbox' \| 'radio' \| boolean` | - |
