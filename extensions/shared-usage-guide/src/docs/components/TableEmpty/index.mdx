### API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| createButton | 创建按钮 | `boolean \| ReactElement` | - |
| createButtonText | 创建按钮文本 | `ReactNode` | - |
| clickCreateButtonFn | 点击创建按钮回调 | `(event: MouseEvent) => void` | - |
| children | 子元素 | `ReactNode` | - |
