### API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| module | 模块名称 | `string` | - |
| detail | 详细信息 | `Record<string, any>` | - |
| enableScale | 是否启用缩放 | `boolean` | - |
| onScale | 缩放回调 | `(replica: number) => void` | - |
| theme | 主题 | `'light' \| 'dark'` | - |
| chartSize | 图表尺寸 | `number` | - |
