### API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| pageSize | 每页大小 | `number` | - |
| cluster | 集群名称 | `string` | - |
| workspace | 工作空间 | `string` | - |
| value | 当前值 | `string` | - |
| onChange | 变化回调 | `(name: string) => void` | - |
